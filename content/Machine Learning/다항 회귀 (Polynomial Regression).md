## 다항 회귀 (Polynomial Regression)

- **비선형**
	- $y = m_1x + m_2x^2 + \cdots + m_nx^n + b$
- **과소 적합**
	- 훈련 데이터 학습이 덜 된 상태
- **과대 적합**
	- 훈련 데이터 학습이 완벽한 상태
	- <span style="border-radius: 5px; color: black; background-color: salmon">하지만 테스트 데이터에 대한 정확도는 낮음</span>
